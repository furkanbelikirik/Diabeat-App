# Diabeat App

Diabeat, diyabet hastaları için geliştirilmiş bir sağlık takip uygulamasıdır.

## Tech Stack

### Frontend

- Next.js App Router
- TypeScript
- Tailwind CSS
- Redux Toolkit
- React Hook Form
- Recharts

### Backend

- Supabase
- Supabase Auth
- PostgreSQL
- RTK Query

### Geliştirme Araçları

- Prettier
- Biomejs
